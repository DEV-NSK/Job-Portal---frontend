## 🚀 Getting Started

### 1. Install Dependencies
```bash
cd frontend
npm install
```

### 2. Start Development Server
```bash
npm run dev
```
Visit: http://localhost:5173/

### 3. Build for Production
```bash
npm run build
```

### 4. Deploy
```bash
# Vercel
vercel

# Netlify
netlify deploy --prod --dir=dist

# Or upload dist folder to your hosting
```
